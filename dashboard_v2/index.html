Mono Output Setup

Hear the whole game through your working ear while the radar still shows direction. AudioRadar does the mono mix for you, so you turn off the Windows "mono audio" setting (leaving it on hides the direction the overlay needs). Follow the 5 steps below once.

Mono plays to
Pick your real headphones here, not the virtual cable.
  1. Install VB-CABLE (button above if shown), then reboot if Windows asks.
  2. Turn off Windows mono audio: Settings › Accessibility › Audio. AudioRadar handles mono now.
  3. Send your game's sound into the cable: Settings › System › Sound › Volume mixer, find your game, set its output to CABLE Input.
  4. Above, set "Mono plays to" to your real headphones.
  5. Pick your game under Program, then press Start.
