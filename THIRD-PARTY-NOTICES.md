# Third-Party Notices

Visual Audio Overlay uses the following third-party components.

## VB-CABLE (VB-Audio Virtual Cable)

The **Mono Output** feature uses VB-CABLE, a virtual audio device, to route a
game's audio to the app so it can be down-mixed to mono for single-sided
listeners without the original being heard twice.

- VB-CABLE is a product of **VB-Audio Software**.
- Origin: https://vb-audio.com/Cable/ (www.vb-cable.com)
- **VB-CABLE is donationware.** It is free to use, and donations are welcome.
  If the Mono Output feature is useful to you, please consider supporting
  VB-Audio: https://vb-audio.com/Cable/ (see the donation links on their site).

VB-CABLE is distributed unmodified. It is a separate product from Visual Audio
Overlay and is not covered by this project's license. All rights to VB-CABLE
belong to VB-Audio Software.
